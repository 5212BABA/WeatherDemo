Missing API Key

This app is unable to obtain weather information without an Open Weather Map API Key. To correct this problem, please do the following:

  1. If you have not already done so, please register with OpenWeatherMap.org
  2. On the How to start page, follow the instructions for creating an API Key
  3. Copy your API Key and paste it as the value of property apikey in the data structure owmQueryParams at line 35 of Utils.scala
  4. Recompile and rerun this app

Scroll and zoom the map to find your desired location, then click that spot to obtain a weather report.
